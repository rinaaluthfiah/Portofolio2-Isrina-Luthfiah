Website portofolio ini dibuat sebagai tugas praktikum mata kuliah Pemrograman Web. Website ini dalam pembuatannya  menggunakan HTML, CSS, serta Bootstrap 5 untuk membantu proses layouting dan responsive design. Tujuan pembuatan website ini   adalah untuk menampilkan profil diri, kemampuan, pengalaman, serta sertifikat dalam bentuk tampilan yang rapi, modern, dan mudah diakses.
  
  Website terdiri dari tiga section utama (wajib), yaitu:
  
   ✦  Home
   
   ✦ About Me
   
   ✦ Certificates

##  🔄 𝐏𝐞𝐧𝐠𝐞𝐦𝐛𝐚𝐧𝐠𝐚𝐧 𝐖𝐞𝐛𝐬𝐢𝐭𝐞 (𝐒𝐭𝐚𝐭𝐢𝐬 → 𝐃𝐢𝐧𝐚𝐦𝐢𝐬)

Pada pengembangan tugas ini, website portofolio yang sebelumnya masih bersifat statis telah diubah menjadi website dinamis menggunakan PHP dan MySQL. Perubahan utama dari pengembangan ini adalah data yang sebelumnya ditulis langsung di dalam HTML (hardcode), kini diambil dari database sehingga lebih fleksibel dan mudah untuk diperbarui tanpa harus mengubah kode program.


  ## 🌟 𝐅𝐢𝐭𝐮𝐫 𝐓𝐚𝐦𝐛𝐚𝐡𝐚𝐧

Website ini memiliki beberapa fitur tambahan, yaitu:

🔹 Website sudah bersifat dinamis, dimana data diambil langsung dari database menggunakan PHP dan MySQL

🔹 Penggunaan fungsi `getData()` untuk mempermudah pengambilan data dan menghindari error jika data kosong

🔹 Data experience dan certificates ditampilkan secara otomatis menggunakan perulangan (looping)

🔹 Gambar dikelola melalui folder khusus (`asets/`) sehingga lebih terstruktur

🔹 Desain responsive dengan bantuan Bootstrap sehingga tampilan tetap rapi di berbagai ukuran layar

## 𝙿𝚎𝚗𝚓𝚎𝚕𝚊𝚜𝚊𝚗 𝙲𝚂𝚂 (𝚜𝚝𝚢𝚕𝚎.𝚌𝚜𝚜)

## ¹ Reset & Global Styling ##

      * {
          margin: 0;
          padding: 0;
          box-sizing: border-box;
          font-family: 'Poppins';
      }
      
      html {
          scroll-behavior: smooth;
      }

⟢ Bagian ini digunakan untuk mereset margin dan padding bawaan browser agar tampilan konsisten.
box-sizing: border-box membuat perhitungan ukuran elemen lebih rapi.
scroll-behavior: smooth membuat perpindahan antar section menjadi halus saat diklik dari navbar.


## ² Navbar Styling ##

      .navbar {
          position: fixed;
          top: 0;
          width: 100%;
          background: rgba(240, 234, 238, 0.85);
          backdrop-filter: blur(20px);
      }

⟢ Navbar dibuat tetap di atas menggunakan position: fixed.
backdrop-filter: blur() memberikan efek blur transparan sehingga terlihat modern.
Hover effect pada .navbar a::after menambahkan garis animasi saat menu disentuh.

## ³ Skills Bar (Progress Bar) ##

    .bar {
        height: 8px;
        background: #eee;
    }
    
    .bar div {
        background: linear-gradient(90deg, #ff7eb3, #ff758c);
    }

⟢ Bagian ini digunakan untuk menampilkan soft skill dalam bentuk progress bar.
Lebar bar diatur melalui width pada HTML untuk menunjukkan persentase kemampuan.


## ⁴ Home Section ##

    .home {
        min-height: 100vh;
        display: flex;
        align-items: center;
    }

⟢ min-height: 100vh membuat section memenuhi tinggi layar.
display: flex dan align-items: center membuat konten berada di tengah secara vertikal.

Efek gradient pada teks menggunakan:

    background-clip: text;
    -webkit-text-fill-color: transparent;
    
yang menghasilkan efek warna gradasi bergerak pada nama

## ⁵ Responsive Design (Media Query) ##

      @media (max-width: 900px) {
          .home-container,
          .about-container {
              flex-direction: column;
          }
      }

⟢ Media query digunakan agar tampilan menyesuaikan ukuran layar.
Saat layar lebih kecil dari 900px, layout berubah menjadi vertikal sehingga tetap rapi di perangkat mobile.
